◆ Generator() [1/2]

core::Generator::Generator ( uint32_t  seed)

Creates the generator for the current thread. All methods of this instance can be accessed by functions from ::core::random. This ensures that no thread-unsafe operations will be performed (because generator will be created for each thread).

NOTE: each thread that wants to use Generator must initialize an instance of this class and keep it alive while it needs its methods.
