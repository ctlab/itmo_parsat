Detailed Description

Gray codes.

A codestruct represents one entry in the code book, i.e. it represents a Gray code of a given length.

For example the Gray code table of length is:

-------------------
|  i  | ord | inc |
-------------------
|  0  |  0  |  0  |
|  1  |  4  |  1  |
|  2  |  6  |  0  |
|  3  |  2  |  2  |
|  4  |  3  |  0  |
|  5  |  7  |  1  |
|  6  |  5  |  0  |
|  7  |  1  |  2  |
-------------------
