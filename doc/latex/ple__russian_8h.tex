P\+LE matrix decomposition of A using Gray codes. 

Returns (P,L,E,Q) satisfying P\+LE = A where P is a permutation matrix of dimension m x m, L is m x r unit lower triangular and S is an r x n matrix which is upper triangular except that its columns are permuted, that is E = UQ for U r x n upper triangular and Q is a n x n permutation matrix. The matrix L and E are stored in place over A.
