\doxysubsection{Detailed Description}
This class solves S\+AT in the following way\+: 


\begin{DoxyItemize}
\item Launches\+Dao multiple \mbox{\hyperlink{classcore_1_1RBSSolve}{R\+B\+S\+Solve}} tasks with the specified configurations, collecting rho-\/backdoors set \{ B\+\_\+1, ..., B\+\_\+s \};
\item For each backdoor set B\+\_\+i it collects the set G\+\_\+i of all assignments for which propagate finds no conflicts;
\item Builds cartesian product G = x \{ G\+\_\+i \};
\item Runs solver\textquotesingle{}s solve for all assignments from G. 
\end{DoxyItemize}
