\doxysubsection{Detailed Description}
Gray codes. 

A codestruct represents one entry in the code book, i.\+e. it represents a Gray code of a given length.

For example the Gray code table of length $2^3$ is\+:

|  i  | ord | inc |

|  3  |  2  |  2  |
|  4  |  3  |  0  |
|  5  |  7  |  1  |
|  6  |  5  |  0  |
|  7  |  1  |  2  |
-------------------
