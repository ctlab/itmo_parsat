\doxysubsection{Detailed Description}
This implementation acts as follows\+: 


\begin{DoxyItemize}
\item Given base assumptions, reduce the current task to a set of tasks.
\item In the resulting set of tasks, solve ones that can be solved fast enough.
\item The rest of problems are put into stack and processed in the same manner.
